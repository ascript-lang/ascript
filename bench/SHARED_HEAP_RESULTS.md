## 1. Headline — shared-heap zero-copy vs per-dispatch deep-clone

Per-dispatch cost of handing a table of N entries to a worker isolate,
two ways: **clone** = pass the plain table (the airlock deep-copies it,
O(N)); **shared** = `shared.freeze` once then pass the `Value::Shared`
(the airlock bumps an `Arc`, O(1)). The worker returns a tiny scalar, so
the time is dominated by argument transport.

| Table size (entries) | Deep-clone / call (ms) | Shared / call (ms) | Speedup |
|----------------------|------------------------|--------------------|---------|
| 5,000 | 0.619 | 0.155 | **4×** |
| 10,000 | 1.059 | 0.144 | **7×** |
| 25,000 | 2.392 | 0.160 | **15×** |
| 50,000 | 4.729 | 0.150 | **31×** |

The shared per-call time is **flat** (one atomic increment regardless of
table size); the deep-clone time grows **linearly** — so the advantage
grows with the table. This is the per-request shared-config win: freeze a
routing table / feature-flag snapshot once, read it across N isolates, pay
zero per-dispatch copy.

### Freeze cost (one-time, amortized)

| Table size (entries) | Freeze (ms) |
|----------------------|-------------|
| 5,000 | 0.22 |
| 10,000 | 0.52 |
| 25,000 | 1.42 |
| 50,000 | 3.01 |

Freeze is O(distinct nodes), paid ONCE; every subsequent dispatch is O(1).

## 2. Gate 12 — no per-read tax on the non-shared hot path

The `Value::Shared` read arm is a single tag check AFTER the existing
Object/Array/Map fast paths, so an ordinary (non-frozen) receiver never
reaches it. `object_churn.as` (6M object create + index/member read) is the
dispatch-bound hot path; its steady state must not regress.

| Measurement | ms |
|-------------|----|
| run 1 | 4962 |
| run 2 | 4963 |
| run 3 | 4858 |
| **median** | **4962** |
| pre-SRV baseline (PROFILING_RESULTS.md) | 4452 |

Delta vs baseline: **+11.5%** → **PASS** (within the ~±15%
machine + build-profile variance band; the added arm is predictably
not-taken for a non-Shared receiver). The three-way `vm_differential` gate
separately proves specialized == generic byte-identically over the corpus,
including the shared examples.
